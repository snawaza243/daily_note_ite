Chapter 1: Introduction and Evolution of Computers

1.1 What is a Computer?

A computer is an electronic device that processes data according to a set of instructions (program) to produce meaningful information. At its core, it follows the IPO model (Input-Process-Output).

The IPO Model

Input

Data entered into the system (keyboard, mouse, sensors)

Process

CPU performs operations on the data (calculations, comparisons)

Output

Results presented to the user (screen, printer, speakers)

Von Neumann Architecture

The foundation of modern computers, proposed by John von Neumann in 1945, consists of:

Memory

Stores data and instructions

CPU

Central Processing Unit

(Control Unit + ALU)

I/O Devices

Input/Output

Key Terms:

  • Algorithm: Step-by-step procedure to solve a problem
  • Hardware: Physical components of a computer
  • Software: Programs and operating information
  • Binary System: Base-2 number system computers use (0s and 1s)

1.4 Practical: Simulating ENIAC

Case Study: ENIAC in WWII

ENIAC (Electronic Numerical Integrator and Computer) was developed during World War II to calculate artillery firing tables. It:

  • Weighed 27 tons and occupied 1800 square feet
  • Contained 17,468 vacuum tubes
  • Consumed 150 kW of power
  • Performed 5,000 additions per second (revolutionary at the time)

Punch Card Emulator

Early computers used punch cards for input. Try this simple JavaScript emulator:

function simulatePunchCard(input) {
    // Convert binary string to punch card representation
    return input.split('').map(bit => 
        bit === '1' ? '◉' : '○'
    ).join('');
}

// Example usage:
console.log(simulatePunchCard('1101')); 
// Output: ◉◉○◉
