What is a Computer?

From Greek 'compute' to modern AI machines

Babbage's Analytical Engine

Charles Babbage's 19th-century "Difference Engine" was the grandfather of your smartphone! His later "Analytical Engine" design incorporated concepts we still use today.

Key Components:

  • Mill: Equivalent to modern CPU's Arithmetic Logic Unit (ALU)
  • Store: Memory capable of holding 1,000 numbers of 50 digits each
  • Reader: Input via punched cards (inspired by Jacquard loom)
  • Printer: Output mechanism for results
  • Control: Conditional branching and loops

Fun Fact: Ada Lovelace wrote the first algorithm for the Analytical Engine, making her the world's first computer programmer.

Von Neumann Architecture (1945)

John von Neumann's design forms the basis of nearly all modern computers with its stored-program concept where both instructions and data are stored in memory.

Key Features:

  • Stored Program Concept: Both data and instructions stored in same memory
  • Sequential Execution: Instructions processed one after another
  • Binary System: Uses base-2 number system for all operations
  • Four Components: CPU, Memory, I/O, and Buses connecting them
